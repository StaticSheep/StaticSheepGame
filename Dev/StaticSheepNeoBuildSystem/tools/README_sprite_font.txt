To generate a new font bitmap run the following from cmd

MakeSpriteFont "Comic Sans" myfile.spritefont /FontSize:16

Further commands

/DefaultCharacter:<value>

-- This character will be used to replace any character in a drawn string that the bmp does not have an image of.
-- If none specified, an exception will be thrown when an attempt to draw a missing character is made
